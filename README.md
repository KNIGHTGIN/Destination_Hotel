# Booking.hotel

## サイト概要
### サイトテーマ
概要：ホテル情報のシェア
テーマ：みんなで情報共有

### テーマを選んだ理由
ホテルに勤めていたから
ホテルの好きな部分のシェアができたら面白いと思ったから

### ターゲットユーザ
観光客、旅行好きな人、旅行気分を味わいたい人

### 主な利用シーン
ホテル選び、旅行先選び、次回の宿泊先候補決め
